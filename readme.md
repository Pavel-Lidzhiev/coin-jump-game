# CoinJump

Простая 2D платформер-игра на React с Canvas.

## Установка

Склонируйте репозиторий и установите зависимости:

```bash
git clone <URL_репозитория>
cd game-app
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

Откроется локальный сервер на `http://localhost:3000`.

## Сборка для продакшена

```bash
npm run build
```

Сборка будет находиться в папке `build`.

## Структура проекта

- `src/` — исходный код игры  
- `App.jsx` — главный компонент  
- `levels.js` — уровни игры  
- `CanvasDisplay.js`, `Player.js`, `Coin.js`, `Lava.js` — логика игры
